# Rules

1. Generate binary files with `.exe` executable even on linux so that we can git ignore them properly.
2. Read root level readme and other files to get quick understandig of the project.
3. This repo contains multiple example apps/libraries that solve one technical problem/challenge.
4. Do not reuse the ports or reassign earlier assigned ports. Always keep incrementing ports. Check ports.txt
5. It is okay to ignore security for now.
6. The task is to solve the problem, fast and crude first.
7. Test and verify the code.
8. Document changes properly in root readme and in the project readme.
9. Build arch.md if needed. Use plantuml and mermaidjs code blocks for diagrams.
